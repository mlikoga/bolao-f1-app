Check the full changelog [here](CHANGELOG.md).

## Development

**Requirements:**
- node v10
- ionic `npm install -g @ionic/cli`
- Firebase CLI (to deploy) `npm install -g firebase-tools`

**To initialize:**
`npm install`

**To run app in local browser:**
`npm start`

**To run tests:**
`npm test`

**To deploy:**

The first time you have to config Firebase, with:
`firebase login`

Then, just run:
`./deploy.sh`
